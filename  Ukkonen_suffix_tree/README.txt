Здесь лежит код построения суффиксного дерева для строки алгоритмом Укконена. 
Он позволяет строить суффиксное дерево в онлайн-режиме, то есть добавлять символ за учетную константу. Суффиксное дерево можно использовать для массы вещей, простейшая из них - поиск подстроки в строке, для которой построено дерево. В качестве демонстрации в main.cpp - функция, в которой можно в цикле на выбор либо расширять имеющийся текст новой строкой, либо проверять наличие в текущем тексте данной подстроки.
