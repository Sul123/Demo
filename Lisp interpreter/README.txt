Здесь лежит код интерпретатора языка Lisp, написанный в рамках выполнения домашнего задания по курсу C++ в ШАДе. 
Требования, по которым написан код, описаны в lispp/readme.md

Из недостатков: не реализован сборщик мусора, поэтому при желании можно организовать утечку памяти, создав цикл из лямбда-функций.
